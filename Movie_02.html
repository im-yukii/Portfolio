Debt Relief Promo

Type: Ads

生成AIを活用し、動画配信者が語りかける形式を採用。広告感を抑えながら自然に視聴を促す構成としました。

媒体・サイズ

TikTok / Meta 広告　（16:9）

ターゲット

借金を抱えている20〜40代男女。配信者の体験談のようなリアルな話であれば、自然と耳を傾けやすい層。

目的

債務整理サービスの新規リード獲得

構成設計

動画配信者が語りかける形式で構成。 冒頭で「500万円近くの借金」という強いワードを提示し、視聴者の注意を即座に引き付ける設計としました。 配信者の独白のような自然なトーンで、ターゲットのリアルなお金事情を赤裸々に語る構成にすることで、広告感を抑えつつ視聴継続を促しています。

クリエイティブ意図

ゲーム実況や配信者のフォーマットは視聴習慣に近く、自然に話を聞いてしまう点に着目 / 視覚的に目に留まりやすいビジュアルで第一印象を強化 / 「借金額」を冒頭提示し、続きを見たくなる設計 / ストーリー性を持たせ、後半でサービス紹介へ接続

デザイン

配信画面を想起させるレイアウトで親近感を演出。 コメント欄風の装飾やUI的要素を取り入れ、広告感を抑制しました。 強調箇所はテロップで整理し、情報の優先順位を明確化しています。

担当範囲・制作期間

リサーチ : 1.5時間企画構成 : 1時間台本作成 : 1時間素材準備 : 1時間デザイン設計 : 1時間編集 : 1.5時間

使用ツール

CapCut / Photoshop / Wan2.2
